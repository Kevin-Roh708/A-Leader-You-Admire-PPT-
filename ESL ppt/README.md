# Bill Gates Presentation (HTML Slide)

이 프로젝트는 Bill Gates에 대한 6장 슬라이드 프레젠테이션을 웹에서 파워포인트처럼 보여주는 HTML 파일입니다.

## 사용법

1. 이 저장소를 클론하거나 파일을 다운로드하세요.
2. `Presentation.html` 파일을 브라우저에서 열면 바로 슬라이드 쇼를 볼 수 있습니다.
3. 슬라이드 이동 방법:
   - 키보드: 좌/우 화살표, Spacebar
   - 마우스: 우측 하단 버튼 클릭
   - 모바일: 좌/우 스와이프

## GitHub Pages로 배포하는 방법

1. 이 저장소를 GitHub에 업로드합니다.
2. 저장소의 Settings > Pages 메뉴로 이동합니다.
3. Branch를 `main`(또는 master)로, 폴더는 `/ (root)`로 선택 후 저장합니다.
4. 잠시 후 `https://your-github-username.github.io/저장소이름/Presentation.html` 주소로 접속하면 웹에서 바로 프레젠테이션을 볼 수 있습니다.

## 파일 구성
- `Presentation.html`: 메인 슬라이드 파일 (이미지 포함, 별도 의존성 없음)
- `README.md`: 이 설명서

## 저작권
- 슬라이드 내용 및 이미지는 예시용이며, 자유롭게 수정/활용 가능합니다.
